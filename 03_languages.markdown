UsbBluetooth provides bindings for multiple programming languages, allowing developers to integrate USB Bluetooth functionality into their projects seamlessly.

## C

The original library is written in C.

- [GitHub Repository](https://github.com/usbbluetooth/usbbluetooth)

## Python

A standalone Python only implementation is available for easy integration into Python applications.

- [GitHub Repository](https://github.com/usbbluetooth/usbbluetooth-python)
- [PyPI Package](https://pypi.org/project/usbbluetooth/)

## C#

A standalone C# only implementation enable .NET developers to use UsbBluetooth in their projects.

- [GitHub Repository](https://github.com/usbbluetooth/usbbluetooth-csharp)
- [NuGet Package](https://www.nuget.org/packages/UsbBluetooth/)
